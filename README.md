

# 🎮 Game Server Dashboard

A professional monitoring dashboard designed for multiplayer game server infrastructure. This project combines my background in Computer Networks with my interest in Game Development, focusing on real-time server performance, connectivity, and player activity.

## 🚀 Features

* Real-time server status monitoring
* Ping / latency display
* CPU and RAM usage metrics
* Active players counter
* Server uptime tracker
* Live system logs
* Responsive modern dashboard UI

## 🛠️ Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript

## 🎯 Purpose of the Project

This project was created to demonstrate how networking concepts can be applied to online gaming environments, especially in multiplayer server infrastructure, monitoring systems, and performance management.

## 📚 Skills Demonstrated

* Backend development with Flask
* API creation and JSON responses
* Frontend dashboard design
* Real-time data updates
* Network monitoring concepts
* Game server infrastructure simulation
